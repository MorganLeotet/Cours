# Les Conditions

*Permettent de faire des choix dans le code

exemple : 

let age = 15

if ( age >= 18 ) {
    console.log(' tu es majeur ! ')
} else {
    console.log(' encore mineur, patience ')
}

traduction : 

si le nombre de la variable age est supérieur ou égal à 18, alors on affiche ' tu es majeur '
sinon si le nombre de la variable est inférieur à 18 alors on affiche ' encore mineur, patience'
