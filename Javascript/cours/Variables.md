# Les Variables

*Une variable sert à stocker une information.
*Il existe 3 façons de créer une variable :

1. let => let.nom = "Alice"   ===> Variable modifiable (qu'on se sert le plus)

2. const => const age = 12    ===> Variable non modifiable

3. var => var ville = "paris" ===> obsolète (qu'on utilise plus)

*pour afficher le résultat d'une variable dans la console (terminal ou navigateur) on utilise :

- console.log()

Exemple : 

let message = 'hello'

console.log(message) // ça va afficher hello dans la console
