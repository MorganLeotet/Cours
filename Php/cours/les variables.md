# Les Variables

une variable = une boite avec une valeur

- Exemple :

$prenom = " ALi ";          prenom contient "ali"
$age = 22;                  age contient 22

==> Afficher une variable

en php ==> echo $prenom;

- Exemple complet :

$nom = "fatou";
echo "Bonjour".$nom;

resultat : "Bonjour Fatou"
