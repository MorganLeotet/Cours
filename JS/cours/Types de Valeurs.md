# Les types de Valeurs

* JS peut stocker différents types de données :

|   Type    |     Exemple    |
|----------------------------|
|   Text (string)    |   "hello"     |
|   Nombre (Number) |       44      |
|   Boolean     |       true / false    |
|   Null        |       Null    |
|   Undefined   |   Undefined   |

## Les Opérateurs

=> Addition : let a = 5 + 3         // 8

=> Soustraction : let b = 10 - 4    // 6

=> Multiplication: let c = 2 * 6    // 12

=> Division: let d = 20 / 5         // 4


*on peut aussi faire modifier une variable

let X = 10

X = X + 2       // 12 (10 + 2)

X += 2          // 14 (12 + 2)

## Les Comparaisons

*Une comparaison renvoie toujours true ou false

- supérieur à  :  5 > 3   renvoie true ( 5 est bien supérieur à 3 )

- inférieur à : 5 < 3     renvoie true ( 5 est bien inférieur à 3 )

- strictement égal à : 10 === 10 renvoie true ( 10 est bien strictement égal à 10 )

- différent de  : 10 != 5 renvoie true ( 10 est bien différent de 5 )
