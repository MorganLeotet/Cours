# Tailwind Css

## C'est quoi ?

Tailwind css est une bilbiothèque Css.

Mais contrairement au Css classique :

    . on n'écrit presque pas de Css
    . on ajoute des classes directement dans le HTML

Exemple Css classique :

button {
    background-color : blue;
    color: white;
    padding: 10px;
}

Même chose avec Tailwind :

<button class="bg-blue-500 text-white p-2">Bouton</button>

Idée clé :  Une class tailwind = une seule règle Css


## Comment penser avec Tailwind

Avec Tailwind :
     - tu ne penses pas : " je crée un bouton "
     - tu penses : " je décris visuellement mon élément "

        => Couleur
        => Taille
        => Espacement
        => Position
