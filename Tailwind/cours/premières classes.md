# Les premières classes

- Les Couleurs :

Exemple : 

<p class="text-red-500">Texte rouge</p>

<div class="bg-green-200">Fond vert clair</p>

    . text- : couleur du texte
    . bg- : couleur du fond


- Les Espacements :

### Padding (espace intérieur)

Exemple :

<div class="p-4">Contenu</div>

    . p => padding
    . p-4 => taille du padding

Variantes : 

    . pt-4 => padding-top
    . pb-4 => padding-bottom
    . px-4 => padding-right
    . py-4 => padding-left


### Margin (espace extérieur)

Exemple :

<div class="m-4">Bloc</div>

Variantes : 

    . mt-4 => margin-top
    . mb-4 => magin-bottom
    . mx-4 => margin-right
    . my-4 => margin-left

- Le Texte

<p class="text-lg font-bold">Bonjour Tailwind</p>
